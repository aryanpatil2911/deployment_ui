Versioning Rules
• There are two types of releases: Major Release and Patch Release.
• Every Patch Release must belong to an existing Major Release.
• Major Release
Recon → 1.MMYY.000
CIDM → 2.MMYY.000
Tools → 3.MMYY.000

• Patch Release
Recon → 1.MMYY.001, .002, .003...
CIDM → 2.MMYY.001, .002, .003...
Tools → 3.MMYY.001, .002, .003...

• Example
Major → 1.0626.000
Patch → 1.0626.001
Patch → 1.0626.002
